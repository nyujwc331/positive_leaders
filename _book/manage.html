6 Project management

Data science projects and applications will be managed and tracked using Jira and GitLab.

6.1 Time

Time spent on specific projects and applications, PTO, and meetings will be tracked in Jira.

6.2 Issues

Tasks will be tracked in GitLab by creating an issue and assigning that issue to a person and project.

6.3 Commit messages

Progress updates will be documented through commit messages.

Follow standard git commit message advice. The first line is the subject, and should summarise the changes in the commit in under 50 characters. If additional details are required, add a blank line, and then provide explanation and context in paragraph format.
